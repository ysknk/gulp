Gulp Starter Kit
====

website build tools.  
use submodule add.  

## Description

* Pug -> html
* Stylus -> css
* js(webpack & Babel) -> js
* img(imagemin) -> img
* other(copy) -> other

## Requirement

* Node.js
* gulp.js(npm global install)

## Install

    git submodule add [this_repo_url] _app
    cd _app/
    node install.js

### before directory

    [root]
      |-[_app]

### after directory

    [root]
      |-[_app]
      |-[_src]
      |-[html]

## Usage

    cd _app/
    gulp
    gulp build
    gulp watch
    gulp lint
    gulp [html|css|js|img]
    gulp [html|css|js|img]:build
    gulp [html|css|js|img]:watch
    .
    .
    .

check other tasks.

    gulp --tasks
